Paper Rock Scissors built in Javascript
=======================================

## Overview

Write up of two player Rock Paper Scissors in Javascript.

## Rules
* rock beats scissors
* scissors beats paper
* paper beats rock
* [same choice] draws with [same choice]

## Tech

- Javascript
- TDD in Jasmine
